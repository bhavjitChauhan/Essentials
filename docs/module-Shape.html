(inner) drawShape(fn, closeopt, modeopt)

Source:
  • essentials.js, line 1634

Alias for beginShape()/endShape().

Examples
drawShape(function() {
    vertex(100, 100);
    vertex(200, 100);
    vertex(200, 200);
    vertex(100, 200);
}, CLOSE);
// expected outcome: square
strokeWeight(5);
drawShape(function() {
    vertex(100, 100);
    vertex(200, 100);
    vertex(200, 200);
    vertex(100, 200);
}, false, POINTS);
// expected outcome: points in a square formation
Parameters:

fn function

Shape function
close CLOSE | boolean <optional>

Whether to close the shape
mode POINTS | LINES | TRIANGLES | TRIANGLE_FAN | TRIANGLE_STRIP | QUADS | QUAD_STRIP <optional>

Shape mode
