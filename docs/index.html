About

The Essentials library provides utility functions for the Khan Academy Processing Environment.

Khan Academy does not provide the equivalent of ES6 Modules or npm for their platform. Users are forced to either reinvent the wheel for themselves or rewrite code in each of their programs.

The Essentials library solves this issue by providing utility functions and constants all packaged in an external file. It allows users to spend more time on their projects than working around Khan Academy's restrictions.

Built With

The Essentials library uses the following open-source projects:

  • jsDelivr
  • Babel
  • Rollup
  • Terser

Getting Started

Create a new program on Khan Academy or use an existing project.

Builder

The recommended method to generate any boilerplate code for the Essentials library is to use the Essentials Builder. Usage information is available on the web application itself.

Content Delivery Network

Essentials is available via the jsDelivr CDN. This option does not increase your program size on Khan Academy at the expense of a slight increase in load times.

Use the following JSHint directive to allow the use of otherwise restricted functionality. See the project Wiki on Khan Academy and JSHint for more information. 

// jshint ignore: start

Use jQuery's getScript method to load the library. 

$.ajaxSetup({ cache: true });
$.getScript('https://cdn.jsdelivr.net/gh/bhavjitChauhan/essentials@1/essentials.min.js',
    function() {
        // your code here
    }
);

See alternative integration methods on the project Wiki page on Usage.

Khan Academy

Essentials is also available locally via the KA build. This code will run as fast as any Khan Academy code. It is recommended to use the Essentials Builder to generate the boilerplate code but alternatively, you may also copy and paste directly from the essentials-ka.min.js file.

Wrapping the Essentials library code in // { and // } allows you to fold the code into one line for aesthetic purposes.

Custom Builds

See the project Wiki page on Custom Builds for customization options to reduce size.

Usage

After loading the Essentials library its functions and constants are defined in the global scope, they can be called directly.

For live examples, please refer to the project Wiki page on Usage.
