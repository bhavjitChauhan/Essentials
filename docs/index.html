The Essentials library provides utility functions for the Khan Academy Processing Environment.

Usage

Create New Program on Khan Academy or add to existing code.

CDN

Use the following JSHint directive to allow the use of otherwise restricted features.

// jshint ignore: start

Use jQuery's getScript method to load the library.

$.ajaxSetup({ cache: true });
$.getScript('https://cdn.jsdelivr.net/gh/bhavjitChauhan/essentials@1/essentials.min.js',
    function() {
        // your code here
    }
);

Local

Essentials is also availible to use without using jsDelivr. Copy and paste the source code directly into your program using the Essentials Builder or the essentials-ka.js file in the repository.

See the project Wiki page on Custom Builds for alternative builds.

Why Essentials?

Khan Academy does not provide the equivalent of ES6 Modules or npm for their platform. Users are forced to either reinvent the wheel for themselves or paste similar lines of code in each of their programs.

The Essentials library solves this issue by providing utility functions and constants all packaged in an external file. It allows users to spend more time on their projects than working around Khan Academy's restrictions.

Documentation is available on Github Pages, the project Wiki or on Khan Academy.
